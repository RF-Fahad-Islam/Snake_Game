# Snake Game üêç

A simple **Snake game** built using **Python and Pygame**. Eat food, grow longer, and try to beat your high score!

## Features

* Classic Snake gameplay
* Score and high score tracking
* Smooth movement using delta time
* Wrap-around screen edges
* Game over screen with restart option

## How to Play

* Arrow keys ‚Üí Move the snake
* Eat red food to grow longer and increase score
* Avoid colliding with yourself

## Installation

1. Clone the repo:

```bash
git clone https://github.com/your-username/snake-game.git
```

2. Install Pygame:

```bash
pip install pygame
```

3. Run the game:

```bash
python snake2.py
```
